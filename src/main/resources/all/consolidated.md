# 🧠 1. CORE DATA STRUCTURES

## Stack
- LIFO
- Ops: push, pop → `O(1)`
- Use: recursion, undo, monotonic stack

## Queue
- FIFO
- Types:
    - Simple Queue
    - Circular Queue
    - Deque
    - Priority Queue (Heap)

## Linked List
- Types:
    - Singly, Doubly, Circular
- Problems:
    - Reverse LL
    - Detect cycle (Floyd)
    - Merge two lists

## Hashing
- HashMap / HashSet → `O(1)` avg
- Patterns:
    - Frequency count
    - Prefix sum

## Heap (Priority Queue)
- Min Heap / Max Heap
- Use:
    - Top K problems
    - Dijkstra

# 🌐 3. GRAPHS

## Traversals
- BFS → Queue
- DFS → Recursion/Stack

- Dijkstra (no negative weights)
- Bellman-Ford (negative allowed)
- Floyd-Warshall (all pairs)
- MST (Prim’s, Kruskal’s)
- Topological Sort (DFS + Kahn’s)

## Patterns
- Grid → Graph
- Multi-source BFS
- Union-Find (cycle detection)
